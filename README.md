# WorthItCheck

WorthItCheck is a static decision-tools site built with plain HTML, CSS, and vanilla JavaScript.

## Live product structure
- `index.html` + `script.js`: main `Repair or Replace` tool
- `tools/`: standalone tool pages and the tools index
- `styles.css`: shared visual system
- `assets/`: brand assets

## Tool expansion system
The process for adding one new tool at a time now lives in [docs/tool-system/README.md](docs/tool-system/README.md).

Start there for:
- the roadmap and next-up queue
- the tool brief template
- the new-tool and improvement-pass checklists
- the scorecard template
- the reusable page scaffold for future tools
- the long-tail cluster workflow for search expansion
